# Parametros del LLM

¿Qué impacto real tiene la temperatura y otros parámetros en una llamada a un modelo?

## Ejercicio


1. Elige un proveedor de referencia. (ex. gemini, openai, huggingface)
2. Busca una API de inferencia. 
    - documenter: a través de la UI, haz una llamada modificando el parámetro. ¿Qué observas? 

    - maker: Escribe un script que se ejecute para, dado un prompt, obtener 5 respuestas con temperaturas diferentes.

    - architect: Diseña, sin implementar, un plan de evaluación para medir el impacto de la temperatura de un golden set de prompts.
